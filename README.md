# Fantasy Ranker Backend

## Run

initiate local env
`cp .env.example .env`

navigate to RIOT GAMES Developer Portal and request your own API KEY. Once you generated your API KEY paste it in the .env.

start server
`make run`

## Endpoints

### health
- GET /health
- GET /version
